## 天气查询

查询城市实时天气，含七日预报、空气质量、生活指数。

| 指令 | 说明 |
|------|------|
| `天气 <城市>` | 查实时天气 |
| `<城市>天气` | 同上 |
| `查天气 <城市>` | 同上 |

**示例：** `天气 上海` / `北京天气`

返回内容：温度、天气状况、湿度、风向风力、气压、日出日落、AQI / PM2.5 空气质量、穿衣 / 防晒 / 运动等生活指数、未来七日预报。

---

## 天气预报

独立七日详细天气卡片。

| 指令 | 说明 |
|------|------|
| `天气预报 <城市>` | 默认 7 天 |
| `天气预报 <城市> <N>天` | 指定天数（1~8） |

**示例：** `天气预报 上海` / `天气预报 北京 3天`

## 健康分析

输入身高、体重、年龄、性别，生成详细健康报告卡片（BMI、体脂率、基础代谢等）。

| 指令 | 说明 |
|------|------|
| `健康分析 <身高> <体重> <年龄> <性别>` | 纯数字格式 |
| `健康分析 身高<X> 体重<X> 年龄<X> <男/女>` | 带单位格式 |

**示例：**
- `健康分析 176 60 24 男`
- `健康分析 身高176 体重60 年龄24 男`

参数范围：身高 50~300cm，体重 10~500kg，年龄 1~150 岁。

## 必应壁纸

获取必应每日精选壁纸及简介。

| 指令 | 说明 |
|------|------|
| `必应壁纸` | 获取今日必应壁纸 |

返回高清 4K 原图 + 标题、简介与版权信息。
