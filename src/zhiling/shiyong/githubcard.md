## **使用**

直接在群聊中发送 GitHub 仓库链接即可自动生成卡片！

**示例：**
```
发送: https://github.com/xxx/xxx
→ 自动生成并发送仓库信息卡片
```

**支持的链接格式：**
- `https://github.com/用户名/仓库名`
- 自动识别，无需其他操作

**卡片样式：**
- Socialify 样式（默认）
- Opengraph 样式

:::tip 💡 提示
自动识别 GitHub 链接，无需手动触发指令
:::
